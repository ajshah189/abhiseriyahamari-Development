# AR Airways

## Project Overview

AR Airways is a luxury digital wedding platform built for the wedding of Abhishek & Riya.

It is not a traditional wedding website.

Instead, the entire wedding is designed as a premium airline-inspired travel experience where every guest becomes a passenger travelling around the world.

The platform combines an interactive resort map, digital passports, boarding passes, AR Miles loyalty program, events, rewards, games and guest engagement into one connected ecosystem.

The objective is to create an unforgettable guest experience that feels more like a premium travel application than a wedding website.

---

# Vision

Every guest should feel like they are travelling.

Instead of simply attending wedding functions, guests progress through a journey.

Each guest receives:

- Digital Boarding Pass
- Digital Passport
- AR Miles Account

Throughout the stay, guests travel across different themed destinations, complete challenges, earn AR Miles, collect passport stamps and unlock achievements.

Every interaction should reinforce the feeling of an airline loyalty experience.

# Core Product Principles

Every feature should strengthen the travel experience.

Avoid isolated pages.

Modules should work together.

Business logic should remain separate from UI.

Code should be modular and reusable.

Never hardcode repeated values.

Prefer configuration over duplication.

Design for scalability beyond a single wedding.

---

# Long-Term Vision

This project should be capable of evolving into a reusable destination wedding platform that can support future weddings with different themes, venues and branding while keeping the same core architecture.
